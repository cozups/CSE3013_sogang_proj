[프로젝트 실행방법]
컴퓨터공학 20172141 김미소

1. 프로그램 실행 후 'l'(소문자 엘) 키를 눌러서 input 파일을 입력합니다.
이 input 파일은 따로 첨부하였으며, 임의로 만들 시에는 x는 0~500 사이, y도 40~500 사이로 지정해주시기 바랍니다.
input 파일은 프로젝트의 bin\data 폴더에 꼭 넣어주시기 바랍니다.

2. 'd' 키를 눌러서 프로그램 실행에 필요한 물체들을 모두 그립니다.

3. 's' 키를 눌러 핀볼 게임을 플레이합니다. 막대 사이 아래에 있는 초록색 박스에 공이 들어가면 게임이 오버됩니다.

4. 게임을 진행하다가 'q'를 눌러 종료할 수 있습니다. 종료 전에는 사용한 메모리들을 해제합니다.
기존의 핀볼 고전게임과 동일하게 'z'키와 '/'키를 눌러서 막대를 움직일 수 있습니다.
공이 막대에 그냥 닿았을 시에는 속도가 느려지고, 막대를 올려 공을 튕겼을 때는 속도가 빨라집니다.

5. 게임을 실행하면 왼쪽 상단에 점수가 표시되며 점수는 막대에 그냥 닿았을 시는 10점, 막대를 올려 공을 튕겼을 때는 300점, 공이 오브젝트에 맞았을 때는 30점이 추가됩니다.

6. 게임오버가 되었을 때에는 'r'키를 눌러 랭킹을 등록할 수 있고 랭킹을 등록하면 프로그램이 종료됩니다.
랭킹파일은 프로젝트의 bin 폴더에 생성됩니다.
